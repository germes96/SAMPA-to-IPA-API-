# XSAMPA API
A simple SAMPA to IPA converter based on an API architecture

## Requiement
1. node js (https://nodejs.org/en/)

## Task

1. Install all dependenciess
```
npm install
```
2. Install nodemon
```
npm install -g nodemon
```
3. Start server
```
nodemon app.js
```
